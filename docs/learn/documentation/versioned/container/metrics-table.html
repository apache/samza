Samza Metrics Reference

The following table lists all the metrics emited by Samza.

Metrics are grouped by group names usually representing classes that are adding their own metrics to the registry. Those are the following:

  • SamzaContainerMetrics
  • job-coordinator
  • TaskInstanceMetrics
  • OffsetManagerMetrics
  • JvmMetrics
  • SystemConsumersMetrics
  • SystemProducersMetrics
  • ContainerProcessManagerMetrics
  • SamzaAppMasterMetrics
  • KafkaSystemConsumerMetrics
  • KafkaSystemProducerMetrics
  • KeyValueStorageEngineMetrics
  • KeyValueStoreMetrics
  • LoggedStoreMetrics
  • SerializedKeyValueStoreMetrics
  • CachedStoreMetrics
  • RoundRobinChooserMetrics
  • BatchingChooserMetrics
  • BootstrappingChooserMetrics
  • HdfsSystemProducerMetrics
  • ElasticsearchSystemProducerMetrics
  • ZookeeperClientMetrics
  • ZkJobCoordinatorMetrics

Words highlighted like this are placeholders for your own variable names defined in configuration file or system variables defined while starting the job.

Note: Average time is calculated for the current time window (set to 300 seconds)
